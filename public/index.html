Agree-Dis Service Api

Agree-dis is a Service that Manage a Debation between 2 Users
- 2 users publish an idea ,people can vote if they agree or not.

This is a documentation for how to use this Service, this documentation contains requests, responses and errors

database Global methods Debates methods
Debates Structure

{
basic_info:{
   title:String,
   img:String,
   voters:[String]
   status:Number
/status can be 0=waiting,1=rejected,2=running,3=closed
},
owner:{
   owner_id:Number,
   owner_votes:Number
},
collaborator:{
   collaborator_id:Number,
   collaborator_votes:Number
 }
}

Users Structure

{
id:Number,
profile:{
email:String,
password:String,
name:{
   first:String,
   last:String
},
age:Number,
imgSrc:String,
},
debates:[String],/*will hold the debates id*/
claps:Number,
vip:Number,
notifications:[String]

Sign-in
'POST' Request
https://agree-dis.herokuapp.com/profile/signIn

Parametrs: { email },{ password } you can't access any of the service route without Signin

Back to Top
on Error: {Error: Signin Failed}
on Success: {Success: you are signed in as { first: ' ', last: ' ' } }
Sign-out
'GET' Request
https://agree-dis.herokuapp.com/profile/signOut

Sign out will Destroy the User Session

Back to Top
Get Debate By Debate ID
'GET' Request
https://agree-dis.herokuapp.com/debates/:debate_id

Visitors can get only the public Debates.
debate owner can Search for any kind of debate.

Back to Top
on Error: {Error: Not authrized} - this Error means that you need to sign in first
on Error: {Error: no Debate Found} - Wrong debate id
on Success: Debate Document

Get all Debates of User
'GET' Request
https://agree-dis.herokuapp.com/debates/userDebate/:usr_id

Visitors can get only the public Debates.
debate owner can Search for any kind of debate,
get all users debate by users ID.

Back to Top
on Error: {Error: Not authrized} - this Error means that you need to sign in first
on Error: {Error: no Debate Found} - Wrong debate id
on Success: Debates Document
Delete Debate
'GET' Request
https://agree-dis.herokuapp.com/debates/deleteDebate/:debate_id

Delete Debate by debate ID
this will also delete all elements linked to this debate from all the database.

Back to Top
on Error:'No Debates Found' - this Error means there is no debate with this id
on Success: {success:'Debate Deleted successfuly'}
Creates new debate
'POST' Request
https://agree-dis.herokuapp.com/debates/createDebate

Creates new Debate.
this will automaticly send notification to the collaborator and wait for response.

Back to Top
on Error: {Error:'ValidationError'}
on Success: new Debate Document
Invitation Response
'GET' Request
https://agree-dis.herokuapp.com/invitationResponse/:debate_id/:response

response to new debate.
if accpeted the debate status will be changed to running,
if rejected the debate status will be changed to rejected.

Back to Top
on Error: {Error:'ValidationError'}
on Error: {Error:'Access Denied'} - the user is not the correct Collaborator
on Success: returns debate id and the status
